Fire Deck. The free, open-source alternative to Pear Deck.

Create presentations with interactivity; these can be accessed using iPads, Chromebooks, and almost any other device using the modern web. To create a presentation, click the Teacher Login button and sign in using Google; to view a pre-made Fire Deck, click on the student button.
